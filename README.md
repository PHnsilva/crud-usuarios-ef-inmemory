# crud-usuarios-ef-inmemory
API REST em C# (.NET 8) com Entity Framework Core e banco InMemory para cadastro de usuários. Implementa CRUD completo com rotas para criar, listar, buscar por e-mail (chave), atualizar (PUT/PATCH) e remover registros. Projeto pensado para fins acadêmicos e testes via Swagger/Postman, com validações básicas e seed inicial de usuário.
